# Hotel Booking Form

A simple and modern hotel booking web application built using FastAPI and PostgreSQL.

Users can fill out a booking form and their booking details are stored in a remote database.

---

# Features

- Hotel booking form
- FastAPI backend
- PostgreSQL database
- Modern frontend design
- Remote database using Supabase
- Deployed on Vercel
- GitHub integration

---

# Technologies Used

- HTML

# How to Run

## Create Virtual Environment

```bash
python -m venv .venv
```

## Activate Environment

### Windows

```bash
.venv\Scripts\activate
```

---

## Install Packages

```bash
pip install -r requirements.txt
```

---

## Run Project

```bash
uvicorn main:app --reload
```

Open in browser:

```bash
http://127.0.0.1:8000
```
